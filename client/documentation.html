Book API Documentation

Get Books - /getBooks - HEAD GET

Query Params

author (string, optional) - An author name to filter by

genre (string, optional) - A genre to filter by

language (string, optional) - A language to filter by

country (string, optional) - A country to filter by

year (number, optional) - A release year to filter by

Returns

A JSON array of book objects

Format: [{author: string, country: string,
language: string, link: string,
pages: number, title: string,
year: number, genres?: [string],
readStatus?: string }]

Examples

Format: [{author: string, country: string, language: string, link: string, pages: number, title: string, year: number, genres?: [string], readStatus?: string}]
