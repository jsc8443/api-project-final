Book API Documentation

Get Books - /getBooks - HEAD GET

Query Params

author (string, optional) - An author name to filter by
genre (string, optional) - A genre to filter by
language (string, optional) - A language to filter by
country (string, optional) - A country to filter by
year (number, optional) - A release year to filter by

Returns

A JSON array of book objects
Format: [{author: string, country: string, language: string, link: string, pages: number, title: string, year: number, genres?: [string], readStatus?: string }]

Examples

Request: /getBooks?author=William%20Faulkner&genre=&language=&country=&year=
Result: [{"author":"William Faulkner","country":"United States", "language":"English","link":"https://en.wikipedia.org/wiki/Absalom,_Absalom!\n", "pages":313,"title":"Absalom, Absalom!","year":1936, "genres":["Southern Gothic","Historical Fiction"]}, ...]

Get Book by Title - /getBook - HEAD GET

Query Params

title (string, optional) - The title of the book

Returns

A JSON book object
Format: [{author: string, country: string, language: string, link: string, pages: number, title: string, year: number, genres?: [string], readStatus?: string }]

Examples

Request: /getBook?title=Beloved
Result: {"author":"Toni Morrison","country":"United States","language":"English", "link":"https://en.wikipedia.org/wiki/Beloved_(novel)\n","pages":321,"title":"Beloved","year":1987}

Get Random Book - /getRandom - HEAD GET

Query Params

author (string, optional) - An author name to filter by
genre (string, optional) - A genre to filter by
language (string, optional) - A language to filter by
country (string, optional) - A country to filter by
year (number, optional) - A release year to filter by

Returns

A JSON book object
Format: [{author: string, country: string, language: string, link: string, pages: number, title: string, year: number, genres?: [string], readStatus?: string }]

Examples

Request: /getRandom?author=&genre=&language=Italian&country=&year=
Result: {"author":"Dante Alighieri","country":"Italy","language":"Italian", "link":"https://en.wikipedia.org/wiki/Divine_Comedy\n","pages":928, "title":"The Divine Comedy","year":1315,"genres":["Epic Poetry","Religious Literature"]}

Get Book Titles - /getTitles - HEAD GET

Body Params

title (string, optional) - The title of the book
author (string, optional) - The author of the book
genre (string, optional) - A genre of the book
language (string, optional) - The language the book is written in
country (string, optional) - The country the book was written in
year (number, optional) - The release year of the book

Returns

A JSON array of book objects
Format: [{author: string, country: string, language: string, link: string, pages: number, title: string, year: number, genres?: [string], readStatus?: string }]

Examples

Request: /getTitles?author=&genre=&language=&country=Russia&year=
Result: ["Stories","Crime and Punishment","The Idiot","The Possessed", "The Brothers Karamazov","Dead Souls","War and Peace","Anna Karenina", "The Death of Ivan Ilyich"]
